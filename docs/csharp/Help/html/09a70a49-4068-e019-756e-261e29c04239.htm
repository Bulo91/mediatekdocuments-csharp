      Utilisateur Propriétés

      Propriétés

      AccesCommandes Obtient ou définit une valeur indiquant si l'utilisateur peut gérer les commandes (JSON : accesCommandes).
      AccesDocuments Obtient ou définit une valeur indiquant si l'utilisateur peut gérer les documents (JSON : accesDocuments).
      AccesExemplaires Obtient ou définit une valeur indiquant si l'utilisateur peut gérer les exemplaires (JSON : accesExemplaires).
      Actif Obtient ou définit une valeur indiquant si le compte est actif (JSON : actif).
      Id Obtient ou définit l'identifiant de l'utilisateur (JSON : id).
      IdService Obtient ou définit l'identifiant du service (JSON : idService).
      LibelleService Obtient ou définit le libellé du service (JSON : libelleService).
      Login Obtient ou définit le nom de connexion (JSON : login).
